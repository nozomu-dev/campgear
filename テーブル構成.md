

# 貸し借り管理

## データ設計

### テーブル関係

グループ：ユーザー        -> 多 対 多
ユーザー：商品           -> 1  対 多
商品：商品カテゴリー      -> 1  対 多
商品：保管場所           -> 1  対 1
保管場所：ユーザー        -> 1  対 1

### テーブル
    - グループ（契約ID, チーム, サークル）
        - グループID
        - グループネーム
    - 所属グループ[中間テーブル]
        - グループID
        - ユーザーID
    - ユーザー
        - ユーザーID
        - ユーザー名
        - 保管場所ID
        - （性別）
        - （年齢）
    - 商品
        - ID
        - 商品名
        - 画像
        - 商品カテゴリーID
        - 商品説明（用途）
        - 色
        - ブランド
        - 在庫数
    - 商品カテゴリー
        - ID
        - 商品カテゴリー名
    - 保管場所
